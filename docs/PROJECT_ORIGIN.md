# Project origin and attribution

Synapse HEA began as a University of Trento Software Engineering group project by:

- Miranda Duraku
- Ritik Kumar Rai
- Meliza Bodurri
- Nomaan Munshi

This repository is an **individual engineering revision maintained by Nomaan Munshi**. It preserves the original project concept while making the submitted implementation coherent enough to study, demonstrate, and continue developing.

The revision focuses on:

- route/controller integration;
- JWT authentication and role authorization;
- demo OTP login;
- appointment conflict handling;
- doctor and medical-record APIs;
- MySQL schema consistency;
- audit logging;
- frontend dashboard integration;
- automated contract checks;
- Docker and CI setup.

The original team contribution should remain credited when this project is discussed publicly. The individual revision should be described as a refactor and extension, not as a solo creation of the original project.
